# 🚆 Train Status Finder

Welcome to **Train Status Finder**, your one-stop solution for tracking train status in real-time.
Get accurate and up-to-date information on train schedules, delays, and cancellations to plan your journey better.

![TrainStatusFinderDemo](https://github.com/KrishBharadwaj5678/TrainStatusFinder/blob/main/LiveTrainStatusFinderDemo.png)

## ✨ Features

| Feature                | Description                                            |
| ---------------------- | ------------------------------------------------------ |
| 🚆 Live Train Status   | Track real-time train location and running status      |
| ⏱️ Delay Information   | Get updates on train delays and expected arrival times |
| ❌ Cancellation Updates | Stay informed about cancelled trains                   |
| 📅 Schedule Details    | View complete train schedules                          |
| ⚡ Fast & Responsive    | Instant updates with a smooth user experience          |
| 🌐 Web-Based           | Accessible from any device via browser                 |

| Technology                | Purpose                              |

| 🐍 **Python**             | Core programming language            |

## 🚀 Getting Started

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/KrishBharadwaj5678/TrainStatusFinder.git
```

### 2️⃣ Navigate to the Project

```bash
cd TrainStatusFinder
```

### 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

### 4️⃣ Run the App

```bash
streamlit run app.py
```
